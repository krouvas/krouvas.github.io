Monopoly Game Analysis     - 12/2019

In this project first a Python script was used for the simulation of a set of rounds on a realistic Monopoly board and to produce useful data.
These data include Priority numbers to help with deciding which property to buy and also the probability of finding a piece at each tile at the end of each round.
Finally, an R script was used to present the probabilities of each tile in a heatmap of the whole Monopoly board as displayed below.

Python Rock Paper Scissors Game     - 02/2019

A simple text-based Rock Paper Scissors game was coded using the Python programming language. I was learning Python on my own and this was my first experience applying it, which helped me greatly improve on it. This program uses Random, or Game Theory based approaches in an attempt to defeat the player and saves scores with the use of a generated save file.
